In PHP 7.4, the ellipsis operator was introduced to unpack arrays. Initially, it only supported integer keys, and not string keys. This was introduced in PHP 8.0.

PHP code
________
.. code-block:: php

   <?php
   
   $array = ['a' => 1];
   
   foo(...$array);
   
   function foo($a) {
   	echo $a;
   }
   
   ?>
